# SpringBOT
This is Spring BOT, Spring say hi :) Spring is a multipurpose discord bot which supports some usefull features, like invite logger, welcome messages, polls, basic modderation commands, and you can even be rewarded COINS for being active in your favourite server.
Spring bot is also modular, supporting the ability to turn features on and off and allow certain roles to do certain things. You can even change the colour of the embeds!

## Commands + Features:
#### Modderation
> these commands also work with IDs

Command | What it does
---|---
Ban <@user> <reason>   | Bans a user </br>
Kick <@user> <reason>  | Kicks a user </br>
Unban <@user> <reason> | Unbans a user </br>
Clear <@user> <reason> | Removes messages in bulk </br>
Warn <@user> <reason>  | Warns a user </br>
Unwarn <@user> <amount>| Removes warns from a user </br>
Warns <@user> <reason> | Shows how many warns a user has </br>
   
#### Automation:
Module | What it does
---|---
Auto Role | Gives people roles automatically when they join your server
Chat Sensor| Removes the big bad naughty words
Welcome Message (also image)| Custom welcome message
Invite Logger| Shows what invite someone used and who created it
Polls| Lets you vote
Private Voice Channel Creator| Makes a personal voice channel thats automatically deleted
  
#### Coins and other fun stuff:
Command | What it does
---|---
Flip | Flips a coin.
Roll | Roles a die.
Coins <@user> | Shows how many coins someone has
Shop | Allows you to buy upgrades with coins
User <@user> | Shows a users upgrades
Send <amount> | Sends someone coins `(can be disabled separately from the rest)`

#### Stuff you should probably know :
This Repositiory is only here for improving the bots code to make it better, but do feel free to use parts of this code in your own bot.
If you want to help make the bot better feel free to join the **[Discord server](https://discord.gg/65mmyX8QEn)** Best part is you don't even need to know how to program !
Spring only stores guild/user/channel/role ID's, warns, coins and settings. </br>
No anonymous data is collected.
